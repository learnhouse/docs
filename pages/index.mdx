# Getting started with LearnHouse

## What is LearnHouse?

LearnHouse is an open source LMS (learning managent system) platform that makes it easy for anyone to provide world class educational content. One of the main features is that it offers a variety of content types: Dynamic Pages, Videos, Documents & more..
