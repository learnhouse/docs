# Platform Overview

This section covers all the features and capabilities of the LearnHouse platform. Whether you are a student, teacher, or administrator, you will find detailed documentation on every aspect of the system here.

## Sections

- **[Courses](/platform/courses)** — Create and manage courses, chapters, activities, collections, and trails.
- **[Editor](/platform/editor)** — A block-based WYSIWYG editor for building rich course content.
- **[Assignments](/platform/assignments)** — Assessable tasks with grading, submissions, and auto-graded code execution.
- **[Playgrounds](/platform/playgrounds)** — Interactive code environments for hands-on practice and experimentation.
- **[Discussions](/platform/discussions)** — Community discussion threads with comments, voting, and reactions.
- **[Boards](/platform/boards)** — Real-time collaborative documents for group work and brainstorming.
- **[Podcasts](/platform/podcasts)** — Audio content with episode management and streaming.
- **[Certifications](/platform/certifications)** — Automatic certificate generation on course completion.
- **[Analytics](/platform/analytics)** — Engagement tracking and usage metrics for your organization.
- **[Search](/platform/search)** — Find courses, collections, and users across your organization.
- **[Payments](/platform/payments)** — Sell courses with Stripe, manage offers, subscriptions, and enrollments.
- **[AI](/platform/ai)** — Context-aware AI assistant for students and teachers.
- **[Users & Access](/platform/users)** — User management, roles, permissions, and authentication.
- **[Organizations](/platform/organizations)** — Multi-tenant organization management, branding, and settings.
